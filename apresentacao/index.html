01

O Problema Físico

Um sistema clássico da mecânica newtoniana: três blocos conectados por cordas ideais passando por uma polia sem massa.

m₁

Bloco 1

Sobre o plano inclinado, conectado por corda ao bloco 2. Sujeito a peso, normal, atrito e tração T₁.

m₂

Bloco 2

Também sobre o plano, entre m₁ e a polia. Sujeito a peso, normal, atrito e trações T₁ e T₂.

m₃

Bloco 3

Pendurado verticalmente pela polia. Seu peso gera a força motriz do sistema.

02

Diagrama de Corpo Livre — m₁

Análise isolada das forças que atuam no bloco 1 sobre o plano inclinado.

Peso P₁ = m₁·g

Decomposto em P₁ₓ = m₁·g·sin α (paralelo) e P₁ᵧ = m₁·g·cos α (perpendicular)

Normal N₁ = m₁·g·cos α

Reação do plano, perpendicular à superfície, anula P₁ᵧ

Atrito f₁ = μ·N₁

Opõe-se ao movimento. Usa μₑ (estático) ou μc (cinético)

Tração T₁

Força da corda que conecta m₁ a m₂, ao longo do plano

03

Diagrama de Corpo Livre — m₂

O bloco intermediário recebe duas trações distintas: T₁ (de m₁) e T₂ (da polia para m₃).

Peso P₂ = m₂·g

Componentes P₂ₓ = m₂·g·sin α e P₂ᵧ = m₂·g·cos α

Normal N₂ = m₂·g·cos α

Reação perpendicular do plano inclinado

Atrito f₂ = μ·N₂

Opõe-se ao sentido do movimento relativo

Tração T₁ (rampa abaixo)

Puxado pelo bloco m₁ no sentido da rampa para baixo

Tração T₂ (rampa acima)

Puxado pela corda que passa pela polia, conectada a m₃

04

Diagrama de Corpo Livre — m₃

O bloco suspenso é o mais simples: apenas seu peso e a tração da corda atuam sobre ele.

Peso P₃ = m₃·g

Força gravitacional atuando verticalmente para baixo

Tração T₂ (para cima)

Força da corda que passa pela polia, suspendendo o bloco

05

Equações do Sistema

O algoritmo central resolve o sistema em etapas lógicas, verificando primeiro o equilíbrio estático.

2 Pesos (P)
P₁ = m₁ · g P₂ = m₂ · g P₃ = m₃ · g
3 Componentes no Plano
P₁ₓ = P₁ · sin α P₁ᵧ = P₁ · cos α P₂ₓ = P₂ · sin α P₂ᵧ = P₂ · cos α
4 Forças Normais (N)
N₁ = m₁ · g · cos α N₂ = m₂ · g · cos α
5 Atrito (Max Estático)
f1_max = μₑ · N₁ f2_max = μₑ · N₂ f_max_total = f1_max + f2_max
6 Sentido do Movimento
F_net = P₃ − (P₁ₓ + P₂ₓ) |F_net| ≤ f_max_total → Equilíbrio |F_net| > f_max_total → Se Move
7 Aceleração
f_total_c = μc · (N₁ + N₂) Se blocos sobem (F_net > 0): a = [P₃ − (P₁ₓ+P₂ₓ) − f_total_c] / (m₁+m₂+m₃) Se blocos descem (F_net < 0): a = [P₃ − (P₁ₓ+P₂ₓ) + f_total_c] / (m₁+m₂+m₃) Atrito sempre se opõe ao movimento
8 Trações
Isolando os blocos (Fr = m·a): Se blocos sobem: T₁ = m₁(a + g·sinα + μc·g·cosα) T₂ = m₃(g − a)
Se blocos descem: T₁ = m₁(g·sinα − μc·g·cosα − |a|) T₂ = m₃(g + |a|)
P Peso (m·g)
N Força Normal
f_max Atrito Estático Max.
f_total_c Atrito Cinético Total
F_net Força Resultante (sem atrito)
T Tração da corda
μₑ / μc Coef. Atrito Estático/Cinético
a Aceleração
06

Motor Físico (calc)

O motor de física lê as variáveis, calcula a força resultante e aplica atrito estático e cinético sem usar bibliotecas externas.

simulador/script.js — Motor Físico Completo 
// ── Física ──
function calc() {
    // 1. Leitura dos inputs e valores default de segurança
    const m1 = parseFloat(dom.m1.value) || 0.1;
    const m2 = parseFloat(dom.m2.value) || 0.1;
    const m3 = parseFloat(dom.m3.value) || 0.1;
    const deg = parseFloat(dom.angle.value) || 30;
    const muE = parseFloat(dom.muE.value) || 0;
    const muK = parseFloat(dom.muK.value) || 0;
    const g = parseFloat(dom.g.value) || 9.81;
    const a = rad(deg), sA = Math.sin(a), cA = Math.cos(a);

    // 2. Decomposição das Forças (Px e Py) nos eixos do plano inclinado
    const P1 = m1 * g, P2 = m2 * g, P3 = m3 * g;
    const P1x = m1 * g * sA, P1y = m1 * g * cA;
    const P2x = m2 * g * sA, P2y = m2 * g * cA;
    const N1 = P1y, N2 = P2y;
    
    // 3. Teste do limite de Atrito Estático Máximo
    const f1_max = muE * N1, f2_max = muE * N2, fT_max = f1_max + f2_max;
    const Fnet = P3 - (P1x + P2x);

    let acc, T1, T2, dir, code, f1E, f2E, f1, f2;

    // 4. Teste Lógico: Sistema entra em equilíbrio ou se move?
    if (Math.abs(Fnet) <= fT_max) {
        // A força não superou o atrito estático -> Equilíbrio (Aceleração Zero)
        dir = 'Equilíbrio'; code = 'eq'; acc = 0;
        T2 = P3;
        f1E = Fnet * m1 / (m1 + m2);
        f2E = Fnet * m2 / (m1 + m2);
        T1 = P1x + f1E;
        f1 = 0; f2 = 0;
    } else {
        // Movimento iminente detectado -> Aplica Atrito Cinético
        f1 = muK * N1; f2 = muK * N2; 
        const fT = f1 + f2;
        
        if (Fnet > 0) {
            // m3 é mais pesado, os blocos sobrem a rampa
            dir = 'Blocos sobem (m₃ desce)'; code = 'up';
            f1E = f1; f2E = f2;
            acc = (P3 - P1x - P2x - fT) / (m1 + m2 + m3);
            T1 = m1 * (acc + g * sA + muK * g * cA);
            T2 = m3 * (g - acc);
        } else {
            // Px dos blocos é maior, eles escorregam rampa abaixo
            dir = 'Blocos descem (m₃ sobe)'; code = 'down';
            f1E = f1; f2E = f2;
            acc = (P3 - P1x - P2x + fT) / (m1 + m2 + m3);
            let absAcc = Math.abs(acc);
            T1 = m1 * (g * sA - muK * g * cA - absAcc);
            T2 = m3 * (absAcc + g);
        }
    }

    // 5. Exporta todos os dados processados para a UI e o Canvas desenharem
    return {
        m1, m2, m3, g, deg, alpha: a, muE, muK, sA, cA,
        P1, P2, P3, P1x, P1y, P2x, P2y, N1, N2,
        f1, f2, f1E, f2E, T1, T2, fT_max,
        acc, Fr: (m1 + m2 + m3) * Math.abs(acc), dir, code
    };
}

Renderização Canvas (drawMain)

O desenho da simulação aplica trigonometria e transformações do Canvas 2D sem bibliotecas gráficas.

simulador/script.js — Engine Gráfica 
// ── Canvas principal ──
function drawMain(r) {
    const W = 900, H = 460;
    const ctx = hiDPI(dom.canvas, W, H);
    ctx.clearRect(0, 0, W, H);

    // 1. Vértices do Triângulo calculados por trigonometria simples
    const alpha = r ? r.alpha : rad(30);
    const hyp = 360;
    const Bx = 600, By = 400;
    const Ax = Bx - hyp * Math.cos(alpha), Ay = By;
    const Cx = Bx, Cy = By - hyp * Math.sin(alpha);

    // Chão
    ctx.strokeStyle = '#2a2a2a';
    ctx.lineWidth = 1;
    ctx.beginPath(); ctx.moveTo(40, By); ctx.lineTo(W - 40, By); ctx.stroke();

    // Triângulo
    ctx.fillStyle = 'rgba(60,60,90,0.15)';
    ctx.strokeStyle = '#3a3a5a';
    ctx.lineWidth = 1.5;
    ctx.beginPath(); ctx.moveTo(Ax, Ay); ctx.lineTo(Bx, By); ctx.lineTo(Cx, Cy); ctx.closePath();
    ctx.fill(); ctx.stroke();

    // Ângulo
    ctx.strokeStyle = '#6c9ef8';
    ctx.lineWidth = 1.2;
    ctx.beginPath(); ctx.arc(Ax, Ay, 35, -alpha, 0); ctx.stroke();
    ctx.fillStyle = '#6c9ef8';
    ctx.font = '500 12px Inter';
    ctx.fillText('α', Ax + 40 * Math.cos(alpha / 2), Ay - 14);

    // Ângulo reto
    ctx.strokeStyle = '#333';
    ctx.lineWidth = 1;
    ctx.beginPath();
    ctx.moveTo(Bx - 12, By); ctx.lineTo(Bx - 12, By - 12); ctx.lineTo(Bx, By - 12);
    ctx.stroke();

    // 2. Posição dos blocos ao longo do eixo rotacionado, sensível à animação
    const bW = 48, bH = 30;
    const t1 = 0.28 + (anim ? animOff * 0.08 : 0);
    const t2 = 0.55 + (anim ? animOff * 0.08 : 0);
    const p = t => ({ x: Ax + (Cx - Ax) * t, y: Ay + (Cy - Ay) * t });
    const pos1 = p(t1), pos2 = p(t2);

    function drawBlock(px, py, col, label) {
        ctx.save(); ctx.translate(px, py); ctx.rotate(-alpha);
        ctx.fillStyle = col;
        rRect(ctx, -bW / 2, -bH, bW, bH, 4); ctx.fill();
        ctx.strokeStyle = 'rgba(255,255,255,0.15)';
        ctx.lineWidth = 1;
        rRect(ctx, -bW / 2, -bH, bW, bH, 4); ctx.stroke();
        ctx.fillStyle = '#fff';
        ctx.font = '600 11px Inter';
        ctx.textAlign = 'center'; ctx.textBaseline = 'middle';
        ctx.fillText(label, 0, -bH / 2);
        ctx.restore();
    }

    drawBlock(pos1.x, pos1.y, '#c05050', 'm₁');
    drawBlock(pos2.x, pos2.y, '#40a098', 'm₂');

    // Cordas
    ctx.strokeStyle = 'rgba(255,255,255,0.3)';
    ctx.lineWidth = 1.5;
    ctx.setLineDash([5, 4]);
    ctx.beginPath(); ctx.moveTo(pos1.x, pos1.y); ctx.lineTo(pos2.x, pos2.y); ctx.stroke();
    ctx.beginPath(); ctx.moveTo(pos2.x, pos2.y);
    ctx.lineTo(Cx - 14 * Math.sin(alpha), Cy - 14 * Math.cos(alpha));
    ctx.stroke();

    // Polia
    const pR = 14;
    ctx.setLineDash([]);
    ctx.beginPath(); ctx.arc(Cx, Cy, pR, 0, Math.PI * 2);
    ctx.fillStyle = '#1a1a2a'; ctx.fill();
    ctx.strokeStyle = '#555'; ctx.lineWidth = 2; ctx.stroke();
    ctx.beginPath(); ctx.arc(Cx, Cy, 2.5, 0, Math.PI * 2);
    ctx.fillStyle = '#777'; ctx.fill();

    // 3. Renderização do Bloco m3 no ar
    const m3X = Cx + pR + 18;
    const m3Off = anim ? animOff * 14 : 0;
    const m3Y = Cy + 95 + m3Off;
    const m3S = 42;
    // Corda descendo...
    ctx.strokeStyle = 'rgba(255,255,255,0.3)'; ctx.lineWidth = 1.5; ctx.setLineDash([5, 4]);
    ctx.beginPath(); ctx.moveTo(Cx + pR, Cy); ctx.lineTo(m3X, m3Y - m3S / 2); ctx.stroke();
    ctx.setLineDash([]);
    // Bloco descendo...
    ctx.fillStyle = '#b8a030';
    rRect(ctx, m3X - m3S / 2, m3Y - m3S / 2, m3S, m3S, 4); ctx.fill();
    ctx.strokeStyle = 'rgba(255,255,255,0.15)'; ctx.lineWidth = 1;
    rRect(ctx, m3X - m3S / 2, m3Y - m3S / 2, m3S, m3S, 4); ctx.stroke();
    ctx.fillStyle = '#fff'; ctx.font = '600 11px Inter';
    ctx.textAlign = 'center'; ctx.textBaseline = 'middle';
    ctx.fillText('m₃', m3X, m3Y);

    // 4. Seta indicativa calculada conforme a direção de 'calc'
    if (r && r.code !== 'eq') {
        const mt = 0.75, mp = p(mt);
        const d = r.code === 'up' ? 1 : -1;
        const dx = (Cx - Ax) / hyp * d, dy = (Cy - Ay) / hyp * d;
        ctx.strokeStyle = ctx.fillStyle = r.code === 'up' ? '#5ab868' : '#d06060';
        ctx.lineWidth = 2.5;
        arrow(ctx, mp.x - dx * 18, mp.y - dy * 18, mp.x + dx * 18, mp.y + dy * 18, 9);
    }

    // Info
    if (r) {
        ctx.fillStyle = '#555'; ctx.font = '500 11px Inter'; ctx.textAlign = 'center';
        ctx.fillText(`α=${r.deg}°   μₑ=${r.muE}   μ𝒸=${r.muK}`, (Ax + Bx) / 2, By + 20);
    }
}

function arrow(ctx, x1, y1, x2, y2, h) {
    const a = Math.atan2(y2 - y1, x2 - x1);
    ctx.beginPath(); ctx.moveTo(x1, y1); ctx.lineTo(x2, y2); ctx.stroke();
    ctx.beginPath(); ctx.moveTo(x2, y2);
    ctx.lineTo(x2 - h * Math.cos(a - 0.4), y2 - h * Math.sin(a - 0.4));
    ctx.lineTo(x2 - h * Math.cos(a + 0.4), y2 - h * Math.sin(a + 0.4));
    ctx.closePath(); ctx.fill();
}
